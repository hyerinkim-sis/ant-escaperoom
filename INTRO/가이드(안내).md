# 🌐 프로젝트 독립 패키지(Web 업로드용) 가이드

이 폴더(`games/INTRO`)는 이제 외부 폴더의 도움 없이 **단독으로 실행 가능한 패키지**가 되었습니다. 그대로 압축하여 인터넷에 올리거나 링크로 공유하실 수 있습니다.

## 📂 업데이트된 파일 구조

- `index.html`, `editor.html`
- `assets/`: **핵심 변경 사항**입니다. 모든 이미지와 배경음악 파일이 이 폴더 안에 복사되었습니다. 외부의 `ART&MUSIC` 폴더를 참조하지 않습니다.
- `css/`, `js/`: 스타일 및 로직 파일

## 🎨 주요 변경 사항 (요청 반영)

1. **내부 참조 시스템**: 모든 파일 참조가 `assets/파일명` 형식으로 변경되었습니다. 업로드 시 `INTRO` 폴더 전체만 올리면 링크가 깨지지 않습니다.
2. **7번 화면 디자인**: 마지막 화면에서 이미지를 제거하고 **검정 배경** 처리했습니다. 이제 "사무실 입구로 가자" 텍스트와 "처음으로" 버튼이 깔끔한 검정 배경 위에 나타납니다.
3. **배경음악 자동 매칭**: `ART&MUSIC` 내에 실제 존재하던 `.mp4` 배경음악 파일들을 `assets/`로 복사하고, 기본 설정(`config.js`)에 연결해 두었습니다.

## 🛠️ 설정 변경 및 적용 방법 (동일)

1. `editor.html`을 엽니다.
2. 설정을 수정하면 이미지 경로나 BGM 경로가 자동으로 `assets/ 파일명` 형식으로 생성됩니다. 
3. 복사한 코드를 `js/config.js`에 붙여넣고 저장하면 즉시 반영됩니다.
