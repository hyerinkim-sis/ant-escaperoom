# 🐜 앤트축제 GitHub 업데이트 가이드

이 파일은 게임 수정 후 GitHub에 빠르고 정확하게 업데이트하는 방법을 기록한 문서입니다.

## 🚀 빠른 업데이트 명령어 (PowerShell)

수정 후 아래 명령어들을 순서대로 실행하세요.

```powershell
# 1. games 폴더로 이동 (이미 폴더 안이라면 생략 가능)
cd "C:\Users\앤트09\Desktop\앤트축제\games"

# 2. 모든 변경 사항 스테이징
git add .

# 3. 변경 내용 기록 (따옴표 안에 설명 입력)
git commit -m "내용 수정: [수정 내용 입력]"

# 4. 서버로 전송
git push origin main
```

## 📁 폴더 구조 가이드

*   `INTRO/`, `SCENE1/`, `SCENE2/`: 각 스토리 씬 폴더
*   `game1_word_search/` ~ `game4_whack_a_rat/`: 각 미니게임 폴더
*   `qrcodes/`: 인쇄용 고해상도 QR 코드 이미지 보관 폴더

## ⚠️ 주의 사항

1.  **이미지 경로**: 인쇄 시트(`QR_인쇄용_시트.html`)에서 이미지를 불러올 때는 반드시 `qrcodes/파일명.png`와 같은 **상대 경로**를 사용해야 GitHub Pages에서 정상적으로 보입니다.
2.  **서브모듈 문제**: 폴더 안에 또 다른 `.git` 폴더가 생기지 않도록 주의하세요. 만약 생겼다면 삭제 후 다시 푸시해야 합니다.
3.  **반영 시간**: `git push` 후 실제 사이트에 반영되기까지 약 1~2분 정도 소요될 수 있습니다.
